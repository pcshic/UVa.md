# 輸入說明

第一行為接下來測資數 {% math T %} ({% math T < 20 %})。每組測資有三個正整數，範圍介於 1000 ~ 10000。

# 輸出說明

每組測資輸出中間的數，格式參考範例輸出。

# 範例輸入

``` text
3
1000 2000 3000
3000 2500 1500
1500 1200 1800
```

# 範例輸出

``` text
Case 1: 2000
Case 2: 2500
Case 3: 1500
```
