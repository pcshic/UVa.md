要用大小為 2 \* 1 的磁磚**貼滿**面積 3 \* n 的矩形共有多少種方法？以下是 n=12 的一種貼法。

{% owl-img images/109/p10918b.gif %}

## Input ##

輸入含有多組測試資料。

每組測試資料一列有一個整數 n ({% math 0\leq{n}\leq{30} %})。

當 n=-1 代表輸入結束。請參考 Sample Input。

## Output ##

對每一組測試資料輸出一列，輸出貼磁磚的方法共有多少種。

## Sample Input ##

	2
	3
	8
	12
	-1

## Sample Output ##

	3
	0
	153
	2131
